# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | ✅                 |
| < 1.0   | ❌                 |

## Release Cycle

- **Latest**: v1.0.0
- **Maintenance**: Latest version receives security patches
- **EOL Policy**: Previous minor versions supported for 3 months after next release

## Reporting a Vulnerability

BetterCF is a browser extension that runs user-submitted content from Codeforces.
Security vulnerabilities are taken seriously.

To report a vulnerability:

1. **Do not** open a public GitHub issue.
2. Email the maintainer at **mhdnazrul511@gmail.com** with details.
3. Include the affected version, a description of the issue, and reproduction steps (if applicable).

You should receive a response within 72 hours. If the issue is confirmed,
a fix will be released as soon as possible, and the reporter may be credited
(if desired) in the release notes.
