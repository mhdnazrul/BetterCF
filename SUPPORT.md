# Support

## Getting Help

- **Documentation**: Read the docs in the [docs/](./docs/) directory.
- **Bug reports**: Open a [GitHub Issue](https://github.com/mhdnazrul/BetterCF/issues/new?template=bug_report.md).
- **Feature requests**: Open a [Feature Request](https://github.com/mhdnazrul/BetterCF/issues/new?template=feature_request.yml).
- **Discussions**: Use [GitHub Discussions](https://github.com/mhdnazrul/BetterCF/discussions) for questions and ideas.

## Before Opening an Issue

1. Check the existing [issues](https://github.com/mhdnazrul/BetterCF/issues) to see if your problem has already been reported.
2. Make sure you're running the [latest version](https://github.com/mhdnazrul/BetterCF/releases).

## Installation Help

- **Chrome / Edge**: Load the `dist/extension/` directory as an unpacked extension.
- **Firefox**: Sign the extension via `web-ext` or use the built `.xpi`.
- **Userscript**: Install the built userscript from `dist/userscript/` via Tampermonkey or Violentmonkey.
